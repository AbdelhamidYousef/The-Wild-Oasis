# PizzaSclice

This is a demo of web application for a hotel management system, built with React, React Router, Redux Toolkit, Tailwindcss, Vite, and EsLint.

**Technologies used:**

- React
- React Router
- React Query
- Supabase
- Styled Components
- Vite
- EsLint
- Other Libraries
  - React Hook Form
  - React Hot Toast
  - React Icons
